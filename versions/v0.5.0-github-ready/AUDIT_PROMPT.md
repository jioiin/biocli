# BioPipe-CLI: AI Code Quality Audit Prompt

**Уважаемый AI-ревьюер (Google Antigravity / Claude / GPT-4)!**

Перед тобой исходный код `BioPipe-CLI` — 100% air-gapped локальной утилиты для генерации биоинформатических пайплайнов. Твоя задача — провести глубокий анализ качества кода (Code Quality Assurance), исключая вопросы безопасности (они покрыты в `SECURITY_AUDIT.md`).

Перед началом анализа обязательно изучи `PROJECT_CONTEXT.md`, чтобы понимать архитектурные ограничения проекта (иммутабельность, плагинная система, изоляция LLM).

## Задачи Аудита

Пожалуйста, проанализируй предоставленные файлы `.py` по следующим 8 категориям:

1.  **Пустые заглушки и обманчивые паттерны (Sniff)**: Найди функции, которые возвращают `pass`, `NotImplemented`, пустые строки или фиктивные данные там, где ожидается реальная бизнес-логика.
2.  **Сложность и читаемость (Quality Workflow Meta)**: Выяви функции с цикломатической сложностью > 10. Предложи рефакторинг сложных методов (`AgentLoop`, `SystemProfiler`).
3.  **Типизация и контракты (Strict Mypy)**: Найди пропущенные аннотации типов, использование `Any` без необходимости, или нарушения `Protocol` контрактов.
4.  **Обработка ошибок (AI Code Auditor)**: Проверь модули на перехват слишком широких исключений (`except Exception:`). Убедись, что все системные ошибки оборачиваются в `BioPipeError`.
5.  **Архитектурные антипаттерны (Vibe Shield)**: Проверь утечку состояния (state leakage), жесткое связывание модулей или нарушение Single Responsibility Principle.
6.  **Тесты (Vibe Check)**: Выяви функции ядра, которые сложно протестировать (hard-to-test code). Предложи моки для них.
7.  **Docstrings и Документация**: Найди публичные классы/функции без исчерпывающих docstrings.
8.  **Производительность**: Проверь циклы, работу со строками и файлами на предмет неэффективного использования RAM/CPU в горячем цикле.
